日本語
#使い方
terminalに'python det_bot.py <log file>'と入力。
<log file>は、今回例としてcowrieで集めたログをおいておく。
するとアクセス時間を時系列順に並べその間隔を計算した結果、botでない（つまり人が直接コマンドを入力した）可能性があるipアドレスのリスト一覧が表示される。
